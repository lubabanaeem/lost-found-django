# University Lost & Found Portal

A simple Django-based web application for reporting and browsing lost and found items within a university.

## Features
- Report lost or found items
- View all listed items
- Search items by name or category
- Backend powered by Django

## Tech Stack
- Python
- Django
- HTML/CSS
- SQLite (default database)

## How it works
Users submit item details through a form, and all entries are stored in the database. Items are displayed in a table with a basic search feature.

## Note
This is a basic demo project built for learning purposes.
